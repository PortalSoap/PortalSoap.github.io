Olá, sou o Lucas, desenvolvedor .NET

Nasci em uma cidade metropolitana do grande ABC, São Bernardo do campo, onde cresci. Já desde a infância, demonstrava grande interesse no processo de montar - e desmontar - coisas. Apesar de simpatizar pelas áreas de engenharia e ensino, encontrei minha grande paixão no desenvolvimento de sistemas, área em que me ocupo atualmente e busco me aperfeiçoar a cada dia. Comecei a programar aplicativos de console utilizando C# e, ao longo do tempo, aprendi outras tecnologias, assim como seus respectivos frameworks e bibliotecas. Atualmente, utilizo C# e .NET para desenvolver soluções web, utilizando também JavaScript e as linguagens de marcação HTML e CSS.

TECNOLOGIAS

Busco adquirir proficiência em um conjunto de tecnologias que me permite desenvolver soluções completas, embora o enfoque do meu trabalho esteja em desenvolver o back-end das aplicações.

ASP.NET Core 6+

Com uma imensa adição de variedades, utilizo o ASP.NET Core como principal framework back-end em minhas aplicaões. Aplicando o padrão MVC, consigo implementar a comunicação entre o cliente e o servidor, ao memso tempo em que organizo a estrutura de projeto.

HTML 5

Sendo a melhor opção atualmente para a criação de estruturas visuais web, utilizo HTML na formação das páginas dos meus projetos. A variedade de elementos presentes em sua semântica, torna possível uma melhor organização do conteúdo da página, tornando-a assim, uma excelente ferramenta.

CSS 3

No que diz respeito a estilização e aspecto de página, utilizo CSS 3 em meus projetos. Suas propriedades de estilo e responsividade a tornam uma excelente ferramenta para a criação de websites atrativos e confortáveis.

JavaScript

Apesar de uma infinidade de usos, utilizo JavaScript com o intuito de obter informações do navegador (cliente). Também é possível modelar elementos visuais mais complexos com o uso de frameworks da linguagem, assim como adicionar interatividade a eles. Este portfolio, por exemplo, faz uso de um pequeno script de rolagem.

GIT e GitHub

Utilizo as tecnologias GIT e GitHub com o intuito de armazenar e versionar o código das soluções que desenvolvo em um ambiente online, o que me permite salvar pontos específicos de desenvolvimento e retornar a eles quando necessário.

Docker

Sendo a ferramenta de conteinerização mais popular atualmente, utilizo o Docker para criar pacotes e alocá-los na nuvem.

PROJETOS

Tive a chance de desenvolver alguns projetos durante meus anos de aprendizado. Abaixo, estão alguns deles:

HTML | CSS

Página Responsiva (Clínica Médica)

Esse foi um dos primeiros projetos que fiz após ter adquirido conhecimento sólido nas linguagens de marcação HTML e CSS. Nele, utilizo a propriedade flex para posicionar os elementos na tela de maneira mais flexível. Também utilizo a propriedade @media para que a página se adeque as dimensões do aparelho.

HTML | CSS

Clone Starbucks Coffee

Neste projeto, reproduzo a página inicial da Starbucks Coffee, buscando reproduzir fielmente sua estrutura. Fiz este projeto depois de aprender a utilizar a propriedade grid. Sua utilização constante torna o uso da propriedade @media menos constante e, no caso deste projeto, até mesmo dispensável.

Renpy

Distribuição de Jogo Visual Novel

Este jogo foi feito durante o curso de Análise e Desenvolvimento de Sistemas, em equipe. Nele, é utilizado o motor de jogo Renpy, que é derivado da linguagem Python. Foi um projeto muito gratificante, embora demorado, dada a necessidade da escolha de diversos componentes visuais e criatividade para a criação da narrativa. A maior dificuldade, no contexto da estrutura de código, foi manipular as várias funções oferecidas pela biblioteca oferecida para que o resultado saísse de acordo com o esperado. A semântica da linguagem, em si, é simples, e não houve tanta dificuldade nesse aspecto.

C#

Jogo de Xadrez (CMD)

Este projeto foi desenvolvido depois de ter adquirido conhecimento sólido de classes em C#. Neste projeto, utilizo diversas classes para representar as peças do tabuleiro, de forma que todas elas possuiam um conjunto de métodos que permitiam sua movimentação específica. Também são criadas diversas classes para gerir o jogo e suas regras, assim como exceções quando elas são quebradas.

ASP.NET Core 6 | HTML | CSS

Listagem de Usuários MVC

Este projeto consiste de um sistema de listagem dos usuários já adicionados a uma lista interna e um sistema simples de cadastro de usuários. Fiz este projeto quando já havia aprendido alguns conceitos do padrão MVC Foi utilizada a template oferecida pelo ASP.NET, que fornece e aplica Bootstrap aos elementos das páginas, então, de certa forma, este projeto também é composto pelo framework Bootstrap, apesar de não ter sido adicionado e utilizado propositalmente.

ASP.NET Core 6 | Entity Framework | SQL Server | HTML | CSS | JavaScript

Implementação de API

Este foi o maior projeto que fiz até o momento. Ele é composto, na verdade, por dois projetos interligados. O primeiro projeto trata-se de uma API Web Disponibilizada em um ambiente online, sendo implementada no segundo projeto, que utiliza o padrão MVC para estabelecer a comunicação servidor-cliente. Ambos os projetos utilizam o Entity Framework e um banco de dados SQL Server para a consulta e adição de informações. A junção destes projetos, representa um website de venda de cursos com elementos de login e cadastro além de operações de notas, possíveis pela API citada.

Para mais informações a respeito dos meus projetos e participações, visite meus perfis.
